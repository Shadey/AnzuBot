AnzuBot 0.1.0
=======

An expanable IRC bot written in Coffeescript

Features
=======
1. Config File
2. Easy Modules
3. Admin Commands
4. Titles for posted webpages

Install
===========
```bash
 git clone https://github.com/Shadey/AnzuBot.git && cd AnzuBot
 npm install
```
Run
===
` coffee AnzuBot.coffee `
